# MiniBalance

Модуль для вывода маленького баланса в Navbar сайта (зависит от шаблона).

Module for displaying a small balance in the Navbar of the site (depends on the template).

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.0.3**
